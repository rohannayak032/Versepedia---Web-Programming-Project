# 🎵 Versepedia — Music Lyrics & Artist Encyclopedia

https://versepedia.vercel.app/
A modern, responsive music lyrics web application built with React and Vite.

Versepedia allows users to explore trending songs, read full lyrics, discover artists, and stay updated with the latest music news — all in one place.

---

## ✨ Features

### 🏠 Home / Featured Page

* Highlights **trending music news**
* Shows latest happenings in the music world
* Displays **trending artists** for quick discovery

### 📊 Charts Page

* Displays **Top 20 trending songs**
* Ranked list format
* Click any song to view lyrics

### 📝 Lyrics Page

* Shows **song title** and **artist name**
* Shows love to your artists by **Liking** the songs in the lyrics page!
* Full lyrics displayed on the left
* Artist details on the right

### 🎤 Artists Page

* Browse curated list of artists
* Dedicated artist profile pages including:

  * Artist photo
  * Biography
  * Stats and metrics
  * Popular songs

### 🔍 Search

* Search for any song directly

### ❤️ Favourites

* Save songs for quick access

### 🌙 Dark Mode

* Toggle between dark and light mode

## 🚀 Installation & Running Locally

```bash

git clone https://github.com/rohannayak032/Versepedia---Web-Programming-Project.git

# Navigate into project
cd Versepedia---Web-Programming-Project

# Install dependencies
npm install

# Start development server
npm run dev
```

Then open:
http://localhost:5173
