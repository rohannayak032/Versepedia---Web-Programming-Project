# 🎵 Versepedia — Music Lyrics & Artist Encyclopedia

> A modern, responsive music lyrics web application built with React and Vite. Versepedia allows users to explore trending songs, read full lyrics, discover artists, and stay updated with the latest music news — all in one place.

---

## 📸 Overview

Versepedia is a feature-rich music lyrics platform designed to give users an immersive experience browsing songs, artists, and music charts. The app features a clean UI with a built-in **dark mode toggle** for a personalized experience.

---

## ✨ Features

### 🏠 Home / Featured Page
- Highlights **trending music news** and the latest happenings in the music world
- Showcases **trending artists** at the bottom of the page for quick discovery

### 📊 Charts Page
- Displays the **Top 20 trending songs** in a ranked list
- Click on any song to navigate directly to its **lyrics page**

### 📝 Lyrics Page
- Shows the **song title** and **artist name** prominently
- Full **lyrics displayed on the left** side of the page
- **Artist details and song information** displayed on the right side for context

### 🎤 Artists Page
- Browse a **curated list of artists**
- Click on any artist to visit their dedicated **Artist Profile Page**, which includes:
  - Artist photo
  - Biography and details
  - Stats and metrics
  - Their most **popular songs**

### 🔍 Search
- Search for any **song or lyrics** directly from the app

### ❤️ Favourites
- **Save your favourite songs** for quick access later

### 🌙 Dark Mode
- Full **dark/light mode toggle** available across the entire website

## 🚀 Getting Started

### Prerequisites
Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v16 or above)
- npm (comes with Node.js)

### Installation & Running Locally

```bash
# 1. Clone the repository
git clone https://github.com/rohannayak032/Versepedia---Web-Programming-Project.git

# 2. Navigate into the project folder
cd Versepedia---Web-Programming-Project

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Then open your browser and go to: **http://localhost:5173**
